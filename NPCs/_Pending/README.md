# Pending NPC Submissions

This folder holds writer-submitted NPC files that are awaiting lead writer review.

Files here are NOT live. They are not visible to the Unreal Dev Tools panel. They do not appear in Obsidian Dataview queries from `NPCs/`.

---

## How to submit

1. Create a subfolder named exactly as the NPC: `_Pending/NPC-Name/`
2. Place two files inside:
   - `NPC-Name.md` — the completed NPC file built from the NPC Template v3.0
   - `NPC-Name_Submission-Form.md` — the completed Submission Form
3. Notify the lead writer that a submission is ready for review

## Review outcomes

**Approved:** Lead writer moves `NPC-Name.md` to `NPCs/NPC-Name.md` and moves the submission form to `_Characters/NPC-Name/Submissions/`. Character folder created if it doesn't exist.

**Returned:** Lead writer completes the Lead Writer Review section of the submission form with specific return reasons. File stays here until revised and resubmitted.
