# NPC Intake — How to Submit a Character Description

Drop your character description into this folder:
`NPCs/_Pending/_Intake/`

Name the file after the NPC, using hyphens for spaces:
```
Factor-Renne-Saul.md
Solta-Byrne.txt
Grove-Keeper-Sennath.md
```

---

## File format — two options

### Option 1 — Plain prose (simplest)
Just write your character description. The watcher reads everything.

```
She runs the Office of Tolerance with the same expression she has used for
nineteen years: professionally neutral, precisely calibrated, giving nothing
away. The Wound Market moves around her the way water moves around a stone...
```

### Option 2 — With a header (more accurate pre-fill)
Add a few lines at the top before a `---` separator:

```
Writer: Jane Smith
Email: jane@example.com
Tier: 3
District: Wound Market
---
She runs the Office of Tolerance with the same expression she has used for
nineteen years...
```

The header fields are optional — include only what you know. The more you provide, the more accurate the stat pre-fill.

---

## What the watcher produces

Within a minute of dropping your file, three documents appear:

**In `NPCs/_Pending/NPC-Name/`:**
- `NPC-Name_Submission-Form.md` — pre-filled form with stat estimates and tell candidates
- `NPC-Name_Submission-Intake.md` — intake document with story thread connections

**In `NPCs/_Characters/NPC-Name/Notes/`:**
- `NPC-Name_Origin-Story.md` — your original prose, preserved exactly

---

## What to do next

1. Review the pre-filled submission form — LOW confidence stats especially
2. Build the full NPC file using `_Templates/NPC-Template-v3.md`
3. Complete all FILL IN sections in the submission form
4. Email `stephenbrentjenkins@gmail.com` with subject:
   `[NPC SUBMISSION] NPC-Name — Your Name — Date`

---

## Tips for better pre-fills

- **More prose = better stats.** Aim for 150+ words.
- **Describe behaviour, not feelings.** "She sets things down before answering" gives the watcher a tell. "She feels suspicious" gives it nothing.
- **Name factions, locations, relationships.** The watcher detects story thread connections from these.
- **Write in any form.** Scene, character sketch, bullet points, stream of consciousness — all work.
